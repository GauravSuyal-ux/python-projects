# Guessing Game (Python)

A beginner-friendly Python project where the computer generates a random number and the player tries to guess it.

## Project Goal

This project is part of my **python-beginner-projects** repository. It helped me practice fundamental Python concepts by building a simple interactive game.

## Concepts Practiced

* Variables
* User Input (`input()`)
* Conditional Statements (`if`, `elif`, `else`)
* Loops (`while`)
* Functions
* Random Module (`random`)
* Comparison Operators
* Basic Game Logic

## Features

* Computer generates a random number.
* User keeps guessing until the correct number is found.
* Displays hints such as:

  * Too High
  * Too Low
* Congratulates the user after a correct guess.

## How to Run

1. Make sure Python is installed.
2. Open a terminal in the project folder.
3. Run:

```bash
python guessing_game.py
```
