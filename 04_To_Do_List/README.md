# Terminal-Based To-Do List (Python)

A simple terminal-based To-Do List application built using Python. This project helps users manage their daily tasks through an interactive menu-driven interface.

It was created as a beginner Python project to practice programming fundamentals such as loops, functions, lists, input validation, and exception handling.

---

## Features

- Add new tasks
- View all tasks
- Edit existing tasks
- Delete individual tasks
- Clear all tasks (with double confirmation to prevent accidental deletion)
- Exit the application gracefully
- Input validation using `try-except`
- Handles empty task lists gracefully
