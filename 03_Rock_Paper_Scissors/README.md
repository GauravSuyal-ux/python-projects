# Rock Paper Scissors 

A simple terminal-based **Rock Paper Scissors** game built using Python.

In this game, the player competes against the computer. The computer randomly selects one option from rock, paper, or scissors, and the program determines the winner based on the game rules.

## Features

- User can choose:
  - Rock
  - Paper
  - Scissors
- Computer generates a random choice
- Automatically determines the winner
- Handles invalid user inputs
- Allows multiple rounds using a replay option
- Simple and beginner-friendly implementation

## Concepts Used

- Variables
- Data types
- User input and output
- Conditional statements (`if`, `elif`, `else`)
- Functions
- Function parameters and return values
- `while` loops
- Input validation
- Python `random` module
- f-strings
