# Simple Calculator

A beginner-friendly calculator built using Python. This project is menu-driven and performs basic arithmetic operations.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Handles division by zero
- Error handling for invalid input
- Runs continuously until the user exits

## Concepts Used

- Functions
- While Loop
- If-Else Statements
- User Input
- Exception Handling (`try-except`)

## How to Run

1. Make sure Python is installed.
2. Open a terminal in this folder.
3. Run:

```bash
python calculator.py
```
